# 🎮 GameTask

Aplicativo de produtividade gamificado desenvolvido em Python com interface gráfica.

O objetivo do projeto é transformar tarefas do dia a dia em um sistema de progressão estilo jogo, onde o usuário ganha XP e sobe de nível ao completar atividades.

---

## 🚀 Funcionalidades

- ✅ Criação de tarefas com recompensa em XP  
- 🎯 Sistema de progressão com nível e experiência  
- ✔️ Conclusão de tarefas com checkbox  
- 💾 Salvamento automático dos dados (JSON)  
- 🔄 Carregamento automático ao abrir o app  
- 🗑️ Opção de resetar progresso (Excluir Player)  
- 🖥️ Interface gráfica com Tkinter  

---

## 🛠️ Tecnologias utilizadas

- Python  
- Tkinter (interface gráfica)  
- JSON (persistência de dados)
